### Notes
- To run a rust program 
     1. Compile using 
     ``` 
     rustc nameOfFileYouWantToCompile
     ```
     2. Then run 
     ```
     ./nameOfFile.exe
     ```
     note:use '.exe' for windows

- The "!" calls a rust macro. Without the exclamation point you're calling the function
