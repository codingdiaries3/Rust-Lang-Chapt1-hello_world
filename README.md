### Notes
1. To run a rust program 
     - First compile using 
     ``` 
     rustc nameOfFileYouWantToCompile
     ```
     - Then run using
     ```
     ./nameOfFile.exe
     ```
     note:use '.exe' for windows

2. The "!" calls a rust macro. Without the exclamation point you're calling the function
